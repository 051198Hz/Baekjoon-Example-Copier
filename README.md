## Introduction
백준의 문제 예제 입출력을 아래와 같은 형식으로 클립보드에 복사합니다.

``` shell
입력
(빈 줄)
출력

입력
(빈 줄)
출력

...(반복)
```

```
# https://www.acmicpc.net/problem/1003
3
0
1
3

1 0
0 1
1 2

2
6
22

5 8
10946 17711
```
